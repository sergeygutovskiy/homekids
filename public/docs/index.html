Authenticating requests

This API is authenticated by sending an Authorization header with the value "Bearer {YOUR_AUTH_KEY}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

Отправте запрос на /users/login, чтобы получить токен

Категории словарей

Получить словарь по категории

requires authentication

Example request:
curl --request GET \
    --get "http://localhost:8000/api/client/v1/dictionary-categories/district/dictionaries" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8000/api/client/v1/dictionary-categories/district/dictionaries"
);

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Обновить компанию авторизованного юзера

requires authentication

Example request:
curl --request PUT \
    "http://localhost:8000/api/client/v1/company" \
    --header "Authorization: Bearer {YOUR_AUTH_KEY}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"Компания 1\",
    \"full_name\": \"Полное название компании\",
    \"owner\": \"Владелец\",
    \"responsible\": \"Ответственный\",
    \"organization_type_id\": 1,
    \"district_id\": 1,
    \"is_has_education_license\": false,
    \"is_has_mdedical_license\": false,
    \"is_has_innovative_platform\": false
}"

const url = new URL(
    "http://localhost:8000/api/client/v1/company"
);

const headers = {
    "Authorization": "Bearer {YOUR_AUTH_KEY}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "Компания 1",
    "full_name": "Полное название компании",
    "owner": "Владелец",
    "responsible": "Ответственный",
    "organization_type_id": 1,
    "district_id": 1,
    "is_has_education_license": false,
    "is_has_mdedical_license": false,
    "is_has_innovative_platform": false
};

fetch(url, {
    method: "PUT",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
